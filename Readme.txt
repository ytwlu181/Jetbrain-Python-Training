Tic_tac_toe.py

This is a Tic Tac Toe game based on project guidance from Jetbrain's training: https://hyperskill.org/projects/82?track=2
The Hard AI is implemented based on Minimax algorithm adopted from: https://github.com/Cledersonbc/tic-tac-toe-minimax

To play this game, Run the game, and the game will ask for initial "Input Command: "
The input commands must contain 3 words, 1st word must be "start" in order to start the game. 2nd word decides 1st player, 3rd word decides 2nd player. player word can be any from the list ["user", "easy", "medium", "hard"]

For example, in order to play as first hand as a user against an easy AI, type in commands: "start user easy". Another instance, if user hope to see an easy AI against a hard AI, type in "start easy hard"

When playing as a user, the user is asked to type in coordinates, which contains to numbers from 1 to 3.
For example, in order to play on the bottom left, user should type in "1 1". Bottom right is "3 1", Top left is "1 3".

To exit the game, simply type in "exit" when is asked for "Input Command:"





To-Do-List.py

This is a To-Do-List based on project guidance from Jetbrain's training: https://hyperskill.org/projects/105?track=2

This program prompts a menu asks for operation, and then stores into a database file named "todo.db"
The program communicates with the database file by reading/writing/deleting, so the user can perform different tasks.
The input time format must be in year-month-date format, such as "2020-12-23"
